## Installation

### Prerequisites
- Node.js (v14+)
- MongoDB (local or Atlas URI)
- Git

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/samvidhan-gyan.git
   cd samvidhan-gyan
2. **Setup Backend**
 cd backend
 npm install
 cp .env.example .env
# Update .env with your credentials

3. **Set up frontend**
 cd ../frontend
 npm install
 cp .env.example .env
# Update .env with your API URL

# Run the application 
 cd../backend
 npm run server

 cd../frontend
 npm start
